# [Github](https://www.github.com)入門教學

## 概念

在多人（2-70人）協作專案時，需要有效管理不同版本和意見。
傳統的 Google doc 版本控制對大型團隊來說較不足夠。
Github 提供更強大的協作平台，基於 Git 版本控制系統。
